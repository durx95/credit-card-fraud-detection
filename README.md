
# 💳 Credit Card Fraud Detection using Machine Learning

## 📌 Project Overview
This project is a Machine Learning-based web application that detects fraudulent credit card transactions using a trained classification model.

The model was trained on an imbalanced dataset and deployed using Streamlit Cloud.

---

## 🚀 Features
- Upload CSV file of transactions
- Automatic preprocessing (Scaling of Time and Amount)
- Fraud prediction output
- Displays number of fraudulent transactions detected

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit
- Joblib
